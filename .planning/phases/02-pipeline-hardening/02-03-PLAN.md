Structure:
```python
"""Incremental ingest of SAM.gov contract/grant opportunities."""

import json
import logging
import time
from datetime import datetime, timezone, timedelta

import httpx

from grantflow.config import SAM_GOV_API_KEY, SAM_GOV_API_BASE
from grantflow.database import SessionLocal
from grantflow.models import Opportunity, IngestionLog, PipelineRun
from grantflow.pipeline.logging import bind_source_logger

logger = bind_source_logger("sam_gov")

SEARCH_ENDPOINT = f"{SAM_GOV_API_BASE}/search"
PAGE_SIZE = 10           # conservative — 10 records per API call
MAX_PAGES = 50           # hard cap: 50 pages × 10 records = 500 records max per run
RATE_LIMIT_PAUSE = 1.0   # seconds between requests
```

**`ingest_sam_gov()` function logic:**

1. Check `SAM_GOV_API_KEY`. If empty, log a warning and return stats with `status="skipped"`.

2. Determine `modified_since` date:
   - Query `PipelineRun` for the most recent row where `source="sam_gov"` and `status="success"`
   - If found, use `completed_at - timedelta(days=1)` as the start (1-day buffer for late-arriving data)
   - If not found (first run), use `datetime.now() - timedelta(days=30)` as the lookback (30 days, not full history — conservative first run to stay within rate limits)

3. Build request params:
   ```python
   params = {
       "api_key": SAM_GOV_API_KEY,
       "postedFrom": modified_since.strftime("%m/%d/%Y"),
       "postedTo": datetime.now(timezone.utc).strftime("%m/%d/%Y"),
       "ptype": "o",  # solicitations/contracts
       "limit": PAGE_SIZE,
       "offset": 0,
   }
   ```

4. Paginate with rate-limit awareness:
   ```python
   for page_num in range(MAX_PAGES):
       params["offset"] = page_num * PAGE_SIZE
       resp = httpx.get(SEARCH_ENDPOINT, params=params, timeout=30)
       if resp.status_code == 429:
           logger.warning("rate_limit_hit", page=page_num, fetched=records_processed)
           break  # stop cleanly, commit what we have
       resp.raise_for_status()
       data = resp.json()
       records = data.get("opportunitiesData", [])
       if not records:
           break  # no more data
       # process records...
       time.sleep(RATE_LIMIT_PAUSE)  # respectful pacing
   ```

5. For each record, map to Opportunity model fields:
   - `id = f"sam_gov_{record['noticeId']}"`
   - `source = "sam_gov"`
   - `title = record.get("title", "")`
   - `opportunity_number = record.get("solicitationNumber", "")`
   - `agency_name = record.get("fullParentPathName", "").split("|")[-1].strip()` (last segment of path)
   - `agency_code = record.get("organizationCode", "")`
   - `post_date = _parse_sam_date(record.get("postedDate"))`
   - `close_date = _parse_sam_date(record.get("responseDeadLine"))`
   - `opportunity_status = "posted" if record.get("active") == "Yes" else "closed"`
   - `description = record.get("description", "")`
   - `raw_data = json.dumps(record)`

6. Upsert using SQLAlchemy's `INSERT ... ON CONFLICT DO UPDATE` pattern (same as other ingesters: session.get() then add/merge). Add `records_added` / `records_updated` counts.

7. Write `IngestionLog` row (for health endpoint compatibility with Phase 1) AND `PipelineRun` row.

8. Return stats dict: `{status, records_processed, records_added, records_updated, records_failed, error}`.

Add helper `_parse_sam_date(value)` that handles SAM.gov date formats: `"YYYY-MM-DDTHH:MM:SS-HH:MM"` → ISO 8601 date. Use `datetime.fromisoformat()` with fallback.
