```python
from abc import ABC, abstractmethod

class BaseStateScraper(ABC):
    source_name: str   # e.g. "state_california"
    state_code: str    # e.g. "ca"

    @abstractmethod
    def fetch_records(self) -> list[dict]:
        """Fetch raw records from the portal. Raise on total failure."""
        ...

    @abstractmethod
    def normalize_record(self, raw: dict) -> dict | None:
        """Normalize one raw record into Opportunity field dict. Return None to skip."""
        ...

    def make_opportunity_id(self, source_id: str) -> str:
        """Returns f'state_{self.state_code}_{source_id}'"""
        ...

    def run(self) -> dict:
        """Execute scrape, normalize, upsert. Returns stats dict."""
        ...
```

    - `CKAN_BASE = "https://data.ca.gov"`
    - `DATASET_ID = "california-grants-portal"` (may need resource_id discovery — see below)
    - `fetch_records()`:
      a. First call `{CKAN_BASE}/api/3/action/package_show?id={DATASET_ID}` to get resource list
      b. Find the first resource with format "CSV" or "JSON" (the datastore resource)
      c. Paginate `{CKAN_BASE}/api/3/action/datastore_search?resource_id={id}&limit=1000&offset=N`
      d. Loop until `offset >= total` or empty batch (see RESEARCH.md CKAN pagination pattern)
      e. Use httpx with timeout=60, respect STATE_SCRAPER_REQUEST_DELAY between pages
    - `normalize_record()`: Map CKAN fields to Opportunity schema:
      - `id` = `make_opportunity_id(str(raw.get("_id") or raw.get("id") or ""))` — CKAN uses `_id` for row number
      - `source` = `self.source_name`
      - `title` = `raw.get("Title") or raw.get("grant_title") or ""`
      - `description` = `raw.get("Description") or raw.get("description")`
      - `agency_name` = `normalize_agency_name(raw.get("Agency") or raw.get("agency_name"))`
      - `agency_code` = slug from agency name: `re.sub(r'[^a-z0-9]+', '_', name.lower()).strip('_')[:50]`
      - `close_date` = `normalize_date(raw.get("Application_Due_Date") or raw.get("deadline"))`
      - `post_date` = `normalize_date(raw.get("Posted_Date") or raw.get("open_date"))`
      - `source_url` = `raw.get("URL") or raw.get("url") or ""`
      - `category` = `"State Grant"`
      - `opportunity_status` = `"posted"`
      - `raw_data` = `json.dumps(raw, default=str)`
      - Return None if title is empty after strip
